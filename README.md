# LIFE 2024

## データ変換

1. [データ変換](https://brainvader.github.io/life-2024/)へアクセスする。
2. LIFE クライアントからコピーしたデータを<名前.txt>形式で保存する。
3. 2 を 1 にアップロードする。
4. save を押して<名前.csv>ファイルをダウンロードする。
